# Usage
<!-- usage -->
```sh-session
$ npm install -g watubeplay-cli
$ watubeplay-cli COMMAND
running command...
$ watubeplay-cli (-v|--version|version)

$ watubeplay-cli --help [COMMAND]
USAGE
  $ watubeplay-cli COMMAND
...
```
